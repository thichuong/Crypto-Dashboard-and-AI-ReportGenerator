Tạo file report.html và report.css, report.js chỉ gồm các section, không tạo thêm file khác 3 file này.

Nội dung của file report.html dùng để tải lên file index.html qua file main.js.

Mỗi heading 2 của file word  là 1 section của file report.html

Các section cập nhật theo file word.

Số section bằng số heading 2 của file word.


Bỏ [cite_start] để khi trình bày trên web dễ nhìn hơn.
File report.html có các section có tiêu đề h2 là phần heading 2 của file word, có icon phía trước.
Ví dụ
    <section id="macro-background">
        <h2><i class="fas fa-landmark icon"></i>Phần I: Bối Cảnh Vĩ Mô và Tâm Lý Thị Trường</h2>
